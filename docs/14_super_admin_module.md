## 1. Overview

The Super Admin Module is the platform administration interface used exclusively by Bizinso team members to manage the entire BizSocials platform, including all tenants, subscriptions, support, and platform configuration.

---

## 2. Access & Authentication

### 2.1 Super Admin Roles

| Role | Description | Permissions |
|------|-------------|-------------|
| **SUPER_ADMIN** | Platform owner/CTO | Full access to everything |
| **ADMIN** | Platform administrators | All except billing settings & critical configs |
| **SUPPORT** | Support team (L2/L3) | View tenants, manage tickets, limited actions |
| **VIEWER** | Analytics/reporting | Read-only access |

### 9.2 Security Monitoring

| Monitor | Alert Threshold | Action |
|---------|-----------------|--------|
| Failed logins | >10 in 5 min | Temporary block |
| API errors | >5% error rate | Alert team |
| Unusual activity | Pattern detection | Flag for review |
| Data export | Any super admin | Log + notify |
| Impersonation | Any use | Log + notify owner |

## 10. API Endpoints

### 10.1 Super Admin API

All super admin APIs require admin authentication and are prefixed with `/admin/api/v1/`.

```
AUTHENTICATION
POST   /admin/api/v1/auth/login
POST   /admin/api/v1/auth/mfa/verify
POST   /admin/api/v1/auth/logout
GET    /admin/api/v1/auth/me

TENANTS
GET    /admin/api/v1/tenants
POST   /admin/api/v1/tenants
GET    /admin/api/v1/tenants/{id}
PATCH  /admin/api/v1/tenants/{id}
POST   /admin/api/v1/tenants/{id}/suspend
POST   /admin/api/v1/tenants/{id}/reactivate
POST   /admin/api/v1/tenants/{id}/terminate
POST   /admin/api/v1/tenants/{id}/impersonate
GET    /admin/api/v1/tenants/{id}/users
GET    /admin/api/v1/tenants/{id}/workspaces
GET    /admin/api/v1/tenants/{id}/usage
GET    /admin/api/v1/tenants/{id}/activity

SUBSCRIPTIONS
GET    /admin/api/v1/subscriptions
GET    /admin/api/v1/subscriptions/{id}
PATCH  /admin/api/v1/subscriptions/{id}
POST   /admin/api/v1/subscriptions/{id}/cancel
POST   /admin/api/v1/subscriptions/{id}/refund

PLANS
GET    /admin/api/v1/plans
```
