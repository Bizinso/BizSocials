## 1. Overview

BizSocials is a self-service platform where tenants configure everything themselves. This document defines the onboarding flows for different tenant types, ensuring each gets a tailored experience while maintaining a streamlined process.

---

## 2. Tenant Types

### 2.1 Tenant Classification

| Type | Code | Description | Typical Size | Primary Use Case |
|------|------|-------------|--------------|------------------|
| **B2B Enterprise** | `B2B_ENTERPRISE` | Large agencies, consulting firms | 50+ employees | Multi-client management |
| **B2B SMB** | `B2B_SMB` | Small-medium businesses | 2-50 employees | Company social presence |
| **B2C Brand** | `B2C_BRAND` | E-commerce, retail, D2C | Varies | Consumer engagement |
| **Individual** | `INDIVIDUAL` | Freelancers, solopreneurs | 1 person | Personal business |
| **Influencer** | `INFLUENCER` | Content creators, KOLs | 1-5 people | Personal branding |
| **Non-Profit** | `NON_PROFIT` | NGOs, educational | Varies | Community engagement |

## 7. Verification Flows

### 7.1 Email Verification

- Sent immediately after signup
- Link expires in 24 hours
- Can resend (max 3 times/hour)
- Blocks access until verified

### 7.2 GST Verification (India)

- Optional but required for GST invoices
- Real-time verification via GST API
- Stores legal name from GST records
- Flags discrepancies

### 7.3 Non-Profit Verification

- Document upload required
- Manual review by Super Admin
- 50% discount applied after verification
- Notification sent upon approval/rejection
